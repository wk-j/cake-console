## Interactive Console for Cake (C# Make)

[![Build Status](https://travis-ci.org/wk-j/cake-console.svg?branch=master)](https://travis-ci.org/wk-j/cake-console)

![](Images/CakeConsole.png)

## Installation

```
> npm install -g  cake-console
```

## Usage

Start console

```
> cake-console
```

Exit 

```
> CTRL + c
```
